## 特記事項
倍率は大学公式の一般選抜結果PDF（2026年度・2025年度）の競争率（受験者数÷合格者数、実質倍率。合格者数は繰り上げ含む）。体育学部は3学科それぞれ学科単位の値、経営・工・文学部は学部合計の値。工学部・文学部は複数学科の合算で、偏差値も学科間で幅がある（工:35.0〜45.0、文:42.5〜50.0）。全学部統一選抜・共通テスト利用選抜の結果は別PDFのため倍率には含まない。経営学部経営学科は2022年度改組の新体制。政治経済・経営・国際・観光・情報通信の5学部は3年次から品川キャンパスへ移動（大学公式キャンパス案内）。ゼミは公式サイトで学部代表と確認できる個別ページが短時間で特定できなかったため省略。
