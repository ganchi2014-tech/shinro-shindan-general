## 特記事項
倍率は大学公式の入試結果一覧PDF（2026年度=e11bcdf0…pdf、2025年度=d0f3da3b…pdf）から一般選抜（A/B/C日程等）の学部内合計で算出した実質倍率（受験者数÷合格者数）。共通テスト利用・併用選抜は含まない。保健医療学部は理学療法学科＋診療放射線学科の合算。合格者数は繰上合格を含む。医療看護学部は附属6病院への就職が8割以上（学部公式キャリアページ）。国際教養学部は医療業界への就職が一定数いるのが特長（学部公式進路実績ページ、業種別で情報通信15%・医療13%・製造9%）。スポーツ健康科学部の2026年度一般選抜は合格者絞り込みで倍率上昇（3.1→5.2）。
