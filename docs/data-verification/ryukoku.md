## 特記事項
倍率は大学公式の入試結果PDF（一般選抜前期日程・本学独自試験のみの方式）の学部合計の受験者数÷合格者数（実質倍率）を算出（小数第1位）。2026年度PDFに2025・2026両年度が掲載。偏差値は河合塾提供データ（パスナビ、2026年7月1日更新）。先端理工学部は2027年4月に理工学部・情報学部・環境サステナビリティ学部へ改組予定のため、2027年度入試向け偏差値は新学部名で掲載されており先端理工学部としての偏差値は非掲載（省略）。就職系統は公式Career Data 2027（2025年度卒業生実績・学部別業種内訳）の上位業種から付与。先端理工の就職は理工学部（旧先端理工）掲載実績を使用。心理学部の就職は旧文学部臨床心理学科実績。総合型選抜は文（伝道者推薦型）・政策・国際・社会・先端理工で実施を確認、他学部は確認できず省略。
