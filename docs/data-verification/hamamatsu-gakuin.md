## 特記事項
偏差値は旺文社パスナビ掲載の河合塾偏差値で、浜松学院大学はBF（ボーダーフリー）のため全学科hensachiを省略。倍率は河合塾Kei-Net掲載の一般選抜入試結果（受験者÷合格者、学科別A・B日程計）で2025・2026両年度（いずれも1.0倍）。2025年に現代コミュニケーション学部→地域共創学部、地域共創学科→地域経営学科、子どもコミュニケーション学科→地域子ども教育学科へ名称変更。入試区分（総合型・学校推薦型・一般選抜・共通テスト利用）は公式入試ガイドで確認。看板ゼミは省略。
