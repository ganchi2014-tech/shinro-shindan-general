## 特記事項
スポーツ学部スポーツ学科の1学部1学科構成で、d02〜d04のコース名はmanifestの旧称。現行公式サイトでは8コース制（学校スポーツ／アスリートコーチング／アウトドア・スポーツ／スポーツ政策・文化／スポーツビジネス・メディア／スポーツパフォーマンス分析／トレーニング科学／健康・スポーツ医科学、出典: https://biwako-seikei.jp/department/）。manifestの「コーチングコース」はアスリートコーチングコース、「スポーツビジネスコース」はスポーツビジネス・メディアコース、「健康スポーツコース」は健康・スポーツ医科学コースに概ね対応。入試はコース別でなく学部一括のため偏差値・入試方式は全行同一。一般入試倍率は大学公式サイトに2025・2026年度の入試結果（志願者・合格者数）の公表ページが確認できず省略（パスナビ掲載の直近公表は2023年度で一般選抜計1.1倍）。偏差値は河合塾ボーダー35.0（A日程）・共通テスト得点率47%（パスナビ）。就職実績は就職率6年連続100%、教員・公務員（警察・消防等）・スポーツ関連企業への就職が特色（出典: https://biwako-seikei.jp/career/results/）。
