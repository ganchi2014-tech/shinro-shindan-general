## 特記事項
倍率は大学公式「入試データ」（https://www.chuo-u.ac.jp/connect/admission/data/ 掲載のippan_data2025.pdf / ippan_data2026.pdf）の学部別選抜（一般方式）合計の公表倍率（受験者数÷合格者数）。中央大は方式が多く（5学部共通選抜・英語外部利用・共テ併用・共テ単独）、方式により倍率が大きく異なる。経済学部は2025年8.8倍→2026年5.4倍と変動大。法学部は2023年に多摩から茗荷谷キャンパスへ移転済み。就職は全学的に公務員（都庁・国税庁等）と金融に強い（出典パスナビ進路ページ、2024-2025卒実績）。総合政策学部の総合型選抜の実施有無は特別入試要項から学部個別に確認できなかったため記載省略。
