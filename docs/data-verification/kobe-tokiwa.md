## 特記事項
【学部改組】看護学科は現在「看護学部看護学科」として独立（保健科学部から改組・2025年度）。manifestのdept_id（kobe-tokiwa-d01=保健科学部看護学科）を維持しつつ現行の看護学部のデータを充当。保健科学部には診療放射線学科（偏差値40.0）もあるがmanifest対象外。【倍率】2026年度は大学公式入試実績（一般選抜前期・中期・後期＋社会人・留学生特別選抜の合計、実質倍率=受験者数÷合格者数）。2025年度はマナビジョン掲載値（実質倍率、一般選抜合計）: https://manabi.benesse.ne.jp/daigaku/school/3724/nyushi/bairitsu/hokenkagaku/ 。教育学部のこども教育はBFのため偏差値省略。【就職出典】教育学部=公式進路ページ（私立保育園38%・小学校教諭等）、保健科学部医療検査=公式進路ページ（病院71%）、看護=スタディサプリ進路career（病院看護師91%）。ゼミは公式サイトで代表的な「看板ゼミ」としての明確な紹介ページが確認できなかったため省略。
