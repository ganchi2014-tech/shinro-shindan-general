## 特記事項
【学部改組】名古屋外国語大学は2027年4月に総合英語学部・世界教養学部・地球社会学部の3学部9学科へ再編予定（https://www.nagoyagaidai.com/reorganization/）。世界共生学科（旧世界共生学部）は2025年4月に現代国際学部グローバル共生学科として改組済み、現代英語学科は2027年に英語キャリア学科へ名称変更予定。河合塾偏差値（パスナビ掲載）は2027年度入試（新学部体制）の値のため、旧学科名とのマッピングを各deptのnoteに記載。【倍率算出】2026年度=大学公式入試結果の学科別一般選抜方式合計（志願者÷合格者）、2025年度=パスナビ掲載の学部別一般選抜合計（志願者÷合格者）。【就職】2025年3月卒就職決定率99.4%、航空業界104名（CA内定39名・中部地区13年連続1位）。メーカー・商社・金融・航空・旅行・ホテル・物流・マスコミ・教育が主要進路（https://www.nagoyagaidai.com/career_support/）。employment_fieldsは全学共通の実績に基づく。【キャンパス】全学部とも日進キャンパス（愛知県日進市岩崎町竹ノ山57、https://www.nufs.ac.jp/outline/access/access/）。【ゼミ】3年次から専門ゼミナール制はあるが、個別の看板ゼミ名称を短時間で特定できる公式掲載が見つからなかったため省略。
