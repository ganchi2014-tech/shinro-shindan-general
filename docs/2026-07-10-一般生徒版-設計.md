# 進路診断（一般生徒版）設計メモ

**日付**: 2026-07-10 / **元アプリ**: ハンドボーラー進路診断（2026-05-26 配布版, 単一HTML）

## 目的
ハンドボーラー進路診断をベースに、部活を問わない一般の高校生向けの大学進路診断アプリを**別アプリ**として作成する。

## 配布形態（元と同じ）
- 単一HTMLファイル `進路診断_一般版.html`。オフライン動作、データは各生徒の端末内のみ（localStorage）、「ホーム画面に追加」でアプリ化。zip配布。

## 削除するもの
1. ハンドボール関連すべて
   - 診断ステップ「ハンドボール継続度」「競技実績」
   - 大学詳細の「ハンドボール」タブ、handballデータカテゴリ
   - 競技レベル・リーグ・強化指定のスコアリング
   - 就職希望の「JHL選手」
2. 指定校推薦機能すべて
   - 指定校一括設定画面、指定校マッチ加点(+18/+8)、`shitei_school_default`データ、localStorageキー

## 大学データ
- 既存93校は流用（handball情報のみ除去）。ただしハンドボール推薦専用の遠隔スポーツ校は削除（削除リストは実装時に確定し報告）
- 地域区分をハンドボールリーグ（関西学生1部等）から地理ベース（滋賀/京都/大阪/兵庫・奈良・和歌山/東海/関東/その他）に再編
- **追加**: 関西人気校＋就職に強い大学 約21校
  - 国公立12: 滋賀大・滋賀県立大・滋賀医科大・京都大・大阪大・神戸大・大阪公立大・京都府立大・京都工芸繊維大・京都教育大・和歌山大・奈良女子大
  - 女子大4: 京都女子・同志社女子・武庫川女子・神戸女学院
  - 就職に強い私立5: 大和・京都薬科・大阪医科薬科・豊田工業・金沢工業
- 新規校は既存スキーマ（生活費・進学実績・入試方式・奨学金・キャンパス周辺・滋賀からのアクセス分）。確定できない数値は「要確認」＋推定バナー表示
- データ精度の免責表示を元より強化（「数値は目安。必ず公式サイトで確認」）

## v1 新機能
1. **国公立入試対応**: 入試方式に「国公立（共通テスト＋二次）」を追加。国公立大学にはentry_methodsを整備し、方式マッチのスコアリングに組み込む
2. **興味分野診断**: 職業未定の生徒向けに「好きな教科・興味分野」ステップを追加（文/理工/情報/医療看護/教育/国際語学/経済経営/芸術/スポーツ/生活栄養など）。学部キーワードマッチでスコア加点
3. **学費・経済条件**: 「学費を抑えたい」オプション → 国公立・自宅通学圏・生活費総額の低い大学に加点
4. **模試基準補正**: 偏差値入力時に模試（進研/河合/駿台/不明）を選択 → 河合基準へ換算してマッチング（進研 −7、駿台 +5 目安、換算値は画面に明示）

## スコアリング（変更後の軸）
基本50点 ＋ 就職マッチ ＋ 興味分野マッチ ＋ 偏差値挑戦度（補正後） ＋ 評定×入試方式 ＋ 地域 ＋ 通学/学費 → S+/S/A/B/C
